# 使用 sqlx-cli 进行数据库迁移

## 安装

```bash
cargo install sqlx-cli
```

## 创建迁移文件

```bash
sqlx migrate add create_users_table
```

## 编辑迁移 sql 文件

## 运行迁移

这个命令会连接到你的数据库（使用 .env 里的 DATABASE_URL），并执行 migrations 文件夹中所有尚未运行的 SQL 文件。

```bash
sqlx migrate run
```

# 创建更新的迁移文件

```bash
sqlx migrate add add_user_fields
```
